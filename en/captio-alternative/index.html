FAQ

Why was Captio discontinued?

Captio was removed from the App Store around 2025 and its servers were shut down. No official reason was given, but its reliance on Gmail API is believed to be a contributing factor.

Can I migrate my Captio data?

Captio had no export feature, so direct data migration is not possible. However, all memos sent through Captio remain in your email inbox. Simply set the same email address in Simple Memo and your new memos will appear alongside your old Captio memos.

Will Simple Memo also shut down suddenly?

Simple Memo uses independent infrastructure (Cloudflare Workers + Resend API) with zero Gmail API dependency, minimizing service disruption risks. It also uses a sustainable freemium business model rather than one-time purchase.

Does it work with non-Gmail addresses?

Yes. You can use any email address as the destination: Gmail, Outlook, Yahoo Mail, iCloud Mail, corporate email, or any other provider.

Can I use it for free?

Yes. After the 7-day free trial (unlimited sends), the Free plan allows 3 sends per day indefinitely. Premium ($2.99/month or $29.99/year) unlocks unlimited sends.

Migration Guide for Captio Users

We address the key concerns when switching from Captio, covering data migration, workflow differences, and feature parity. The bottom line: the migration barrier for Captio users is extremely low, and in many ways, switching is an upgrade rather than a compromise.

Data Migration: Nothing to Do
Every memo you sent through Captio is already sitting in your email inbox. Captio was never a storage app -- it was a send-to-email app. Your entire memo history lives in your email, not in the app. This means there is literally no data migration step. Simply install Simple Memo, set the same destination email address you used with Captio, and you are done. New memos will land in the same inbox alongside your old Captio memos, maintaining a seamless chronological timeline. No export files, no import wizards, no data conversion -- just set your email and start writing.
Workflow Comparison: Fewer Steps to Send
Captio's workflow was three steps: open the app, type your text, tap send. With Simple Memo, you can tap the home screen widget to jump straight into the input screen, type your memo, and tap send. Via the widget, the flow is "widget tap, type, send" -- the same as Captio or even smoother. But here is where things get better: Simple Memo adds offline support that Captio never had. You can write and hit send in the subway, on a plane, or in any area with no signal. Your memo is encrypted and stored in a local Outbox, then automatically sent when connectivity returns. You never have to worry about losing a thought because of poor reception -- a major upgrade over the Captio experience.
Feature Parity Checklist
What stays the same as Captio: one-tap send, email delivery, minimal UI, auto-clear after sending, and dead-simple setup with no unnecessary configuration. These core experience pillars are fully preserved in Simple Memo. What is new beyond Captio: AES-GCM encryption (your memo is encrypted on-device before it ever leaves your phone, protecting privacy across the entire transmission path), offline queue (Outbox architecture with automatic resend when back online), send history (browse a list of previously sent memos right inside the app), and dark mode (full support for iOS system-level dark mode). In short, everything Captio could do is covered, plus you gain meaningful improvements in four areas: security, offline reliability, history management, and visual design.

Why Captio Shut Down and Lessons Learned

Captio was a beloved app for many years, but two fundamental challenges -- technical dependency and business model sustainability -- ultimately made it impossible to continue. Simple Memo's architecture was designed with these lessons in mind.

Gmail API Dependency and Its Risks
Captio relied on Gmail's API to send memos as emails. However, Google frequently changed its API policies, progressively tightening security requirements for third-party Gmail access. Changes to OAuth authentication requirements, stricter API scope limitations, and more rigorous app review processes compounded over time, creating an enormous compliance burden for a small development team. Simple Memo learned from this and built a completely independent sending infrastructure with zero Gmail API dependency. The backend runs on Cloudflare Workers, and emails are dispatched via the Resend API. This architecture is not affected by any single email provider's policy changes. You can freely set any email address as your destination -- Gmail, Outlook, Yahoo Mail, iCloud, corporate email, or any other provider.
A Sustainable Business Model
Captio used a one-time purchase model -- pay once, use forever. While attractive for users, this left developers without ongoing revenue to cover server costs and API maintenance, making long-term sustainability difficult. Since sending emails requires continuous server resources, the one-time model created a structural problem where more users meant greater losses. Simple Memo adopts a freemium model instead. The free plan (3 sends per day) provides unrestricted access to all core features, while the Premium plan ($2.99/month or $29.99/year) unlocks unlimited sends for power users. This recurring revenue stream covers server operating costs, security updates, and ongoing feature development in a sustainable way. A 7-day free trial with unlimited sends is also available, so you can experience the full product before deciding whether to subscribe.
