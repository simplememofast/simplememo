生産性メソッド用語 Productivity Method Terms

📭
Inbox Zero Inbox Zero
Merlin Mannが提唱した受信トレイゼロの方法論。メールをタスクキューとして扱う考え方。 Merlin Mann's methodology for keeping your inbox empty. Treating email as a task queue.
詳しく見る → Learn more →
📋
GTD GTD
David AllenのGetting Things Doneシステム。キャプチャ→明確化→整理→レビュー→実行の5ステップ。 David Allen's Getting Things Done system. Five steps: Capture, Clarify, Organize, Reflect, Engage.
詳しく見る → Learn more →
🗃️
ツェッテルカステン Zettelkasten
Niklas Luhmannのスリップボックス式ノート術。一時メモ→文献メモ→永久メモの3段階。 Niklas Luhmann's slip-box note system. Fleeting notes, literature notes, and permanent notes.
詳しく見る → Learn more →
🧠
セカンドブレイン Second Brain
Tiago ForteのBuilding a Second Brain / PARAメソッド。デジタル知識管理の体系的アプローチ。 Tiago Forte's Building a Second Brain and PARA method. A systematic approach to digital knowledge management.
詳しく見る → Learn more →
🍅
ポモドーロ・テクニック Pomodoro Technique
Francesco Cirilloの時間管理テクニック。25分集中+5分休憩のサイクルで生産性を向上。 Francesco Cirillo's time management technique. 25-minute focus + 5-minute break cycles for peak productivity.
詳しく見る → Learn more →
✉️
Captioメソッド Captio Method
「自分にメールでメモを送る」コンセプト。2009年のCaptioアプリに始まる歴史と哲学。 The "send memo to yourself by email" concept. History and philosophy originating from the Captio app (2009).

知識管理・学習用語 Knowledge & Learning Terms

📝
Markdown Markdown
プレーンテキストで書式を表現する軽量マークアップ言語。多くのメモアプリが対応するデファクトスタンダード。 A lightweight markup language for formatting plain text. The de facto standard supported by most memo apps.
詳しく見る → Learn more →
🧠
PKM（個人知識管理） PKM (Personal Knowledge Management)
個人が知識を収集・整理・活用するための体系的なアプローチ。セカンドブレインやZettelkastenの上位概念。 A systematic approach to collecting, organizing, and leveraging personal knowledge. The overarching concept behind Second Brain and Zettelkasten.
詳しく見る → Learn more →
🔁
スペースド・リペティション Spaced Repetition
間隔を空けて繰り返し学習する記憶術。忘却曲線に基づき最適なタイミングで復習することで記憶を定着させる。 A memorization technique using spaced review intervals. Based on the forgetting curve, it optimizes timing to maximize retention.
詳しく見る → Learn more →
🗺️
マインドマップ Mind Map
中心テーマから放射状にアイデアを広げる思考整理法。Tony Buzanが体系化した視覚的なノートテイキング手法。 A visual thinking tool that radiates ideas from a central theme. Systematized by Tony Buzan as a visual note-taking method.
詳しく見る → Learn more →
📓
コーネル式ノート Cornell Notes
コーネル大学で開発されたノートテイキングシステム。ノート・キュー・サマリーの3分割で効果的な学習を実現。 A note-taking system developed at Cornell University. Divides the page into notes, cues, and summary for effective learning.
詳しく見る → Learn more →

集中力・生産性用語 Focus & Productivity Terms

📵
デジタルデトックス Digital Detox
デジタルデバイスやSNSから意図的に距離を置くこと。注意力の回復と深い思考を取り戻すための実践法。 Intentionally disconnecting from digital devices and social media. A practice to restore attention and reclaim deep thinking.
詳しく見る → Learn more →
🎯
ディープワーク Deep Work
Cal Newportが提唱した、集中した状態で認知的に要求の高い作業を行う能力。シャローワークとの対比で理解する。 Cal Newport's concept of performing cognitively demanding work in focused concentration. Understood in contrast with shallow work.
詳しく見る → Learn more →
📊
アイゼンハワー・マトリクス Eisenhower Matrix
緊急度と重要度の2軸でタスクを4象限に分類する優先順位フレームワーク。第2象限（重要だが緊急でない）に集中する方法。 A prioritization framework that classifies tasks into 4 quadrants by urgency and importance. Focus on Quadrant 2 (important but not urgent).
詳しく見る → Learn more →
📌
カンバン Kanban
トヨタ生産方式から生まれた視覚的なワークフロー管理手法。「To Do」「Doing」「Done」のボード形式でタスクを可視化。 A visual workflow management method born from Toyota Production System. Visualize tasks on boards with "To Do," "Doing," and "Done" columns.
詳しく見る → Learn more →
⏱️
タイムボクシング Timeboxing
タスクに固定の時間枠を割り当てる時間管理テクニック。締め切り効果を活用して集中力と完了率を高める。 A time management technique that assigns fixed time blocks to tasks. Leverages deadline pressure to boost focus and completion rates.
詳しく見る → Learn more →
