セカンドブレインとはWhat is Second Brain?

セカンドブレイン（Building a Second Brain）は、Tiago Forteが提唱したデジタル知識管理システムです。脳の外に「第二の脳」を構築し、PARAメソッド（Projects・Areas・Resources・Archives）で情報を体系的に整理。情報を「保存する」のではなく「活用できる形で蓄積する」ことが核心です。 Building a Second Brain is a digital knowledge management system by Tiago Forte. It creates an external "second brain" organized with the PARA method (Projects, Areas, Resources, Archives). The key is not just storing information, but accumulating it in a form ready for future use.

定義

セカンドブレイン（Building a Second Brain）は、Tiago Forteが2022年の著書で体系化したデジタル知識管理手法です。日常的に触れるすべての情報（記事、動画、会話、アイデア）から「将来の自分に役立つもの」を選別し、PARAの4カテゴリで整理します。CODE（Capture・Organize・Distill・Express）の4ステップでインプットをアウトプットに変換し、知的生産性を最大化します。

Definition

Building a Second Brain is a digital knowledge management methodology systematized by Tiago Forte in his 2022 book. It selects "future-useful" information from everything you encounter daily (articles, videos, conversations, ideas) and organizes it into four PARA categories. The CODE workflow (Capture, Organize, Distill, Express) transforms inputs into outputs, maximizing intellectual productivity.

PARAメソッドの4カテゴリThe Four PARA Categories

ProjectsProjects
期限のある進行中のプロジェクト。「ブログ記事を書く」「引っ越し準備」など、完了条件が明確なもの。最も頻繁にアクセスするカテゴリ。Active projects with deadlines. "Write blog post," "prepare for move" — items with clear completion criteria. The most frequently accessed category.
AreasAreas
継続的な責任領域。「健康管理」「財務」「マネジメント」など、終了条件がなく一定水準を維持し続けるもの。Ongoing responsibilities without end dates. "Health," "Finances," "Management" — things requiring sustained standards rather than completion.
ResourcesResources
将来使うかもしれない情報。興味のあるトピック、参考記事、テンプレートなど。プロジェクトに昇格する可能性がある素材。Information for potential future use. Interesting topics, reference articles, templates. Materials that may be promoted to Projects.
ArchivesArchives
完了したプロジェクト、非アクティブなリソース。削除はしないが日常的にはアクセスしない。検索で必要な時に取り出せるように保管。Completed projects, inactive resources. Not deleted but not accessed daily. Stored for retrieval via search when needed.

よくある質問（セカンドブレイン）FAQ: Second Brain

セカンドブレインとは何ですか？

Tiago Forteが提唱したデジタル知識管理システムです。PARAメソッドで情報を4カテゴリに分類し、CODEワークフローでインプットをアウトプットに変換します。

PARAメソッドとは？

Projects（進行中プロジェクト）、Areas（継続的な責任領域）、Resources（将来使う情報）、Archives（完了・非アクティブ）の4カテゴリで全情報を整理する方法です。

What is Second Brain?

A digital knowledge management system by Tiago Forte. It organizes information into four PARA categories and uses the CODE workflow to turn inputs into outputs.

What is the PARA method?

PARA: Projects (active with deadlines), Areas (ongoing responsibilities), Resources (future reference), Archives (completed/inactive). All digital information goes into one of these four categories.
