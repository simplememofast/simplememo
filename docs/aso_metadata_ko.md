**0.3초 실행. 한 번 탭. 내 이메일로 메모 전송.**

SimpleMemo는 떠오른 생각을 가장 빠르게 기록하는 방법입니다. 2025년 서비스를 종료한 Captio의 완벽한 대안으로, 더 빠르고 더 안전하게 만들었습니다.

**왜 SimpleMemo인가요?**

- 0.3초 실행 속도 — 어떤 메모 앱보다 빠릅니다

- 오프라인 지원 — 지하철에서도 메모하면 자동으로 재전송
- 매일 3회 무료 — 광고 없이 영원히 무료 사용 가능

**이런 분들께 딱 맞습니다:**

- 회의 중 떠오른 아이디어를 즉시 기록하고 싶은 분

카카오톡을 열고, 나에게 보내기를 찾고, 채팅방에서 메모를 입력하는 동안 SimpleMemo는 이미 전송을 마칩니다.

### その市場固有の注意点
- **카카오톡 「나에게 보내기」가 사실상 최대 경쟁자.** 한국 사용자 대부분이 카카오톡으로 셀프 메모를 하고 있음. "카톡보다 빠르다"를 핵심 메시지로 사용.
- **네이버 메모** 역시 널리 쓰이나 이메일 전송 기능은 없음. 받은편지함 기반 워크플로우를 강조하면 차별화 가능.
- 한국 App Store에서 「메모」 키워드는 경쟁이 매우 치열함. 「이메일 메모」「셀프 메일」「내게 쓰기」등 롱테일 키워드 공략이 효과적.
- Captio 한국 사용자는 적었으나 IT/스타트업 종사자 중 인지도 있음. 「캡티오 대안」 키워드는 니치하지만 전환율 높음.
- 한국 시장은 구독 요금에 민감. 무료 3회/일을 강조하고, 연간 요금의 할인율을 부각하는 것이 효과적.
- 한국어 리뷰와 별점이 매우 중요. 초기 리뷰 확보를 위한 인앱 리뷰 요청 타이밍 최적화 필요.
