### Step 1（3分）: Tier 1 = 必須3件

GSC上部の検索窓（URL検査）に1件ずつ貼り付け → 検査完了を待つ → **「インデックス登録をリクエスト」** をクリック。1件あたり15〜30秒。


## やらないこと（重要）

- **「ページのインデックス登録」レポートの「検証を開始」ボタンは押さない**（リダイレクト79件・404 7件のバケットは仕様上の表示でエラーではない。404のうち6件は外部で捏造されたURLで、404のままが正解）。
- **「リンクの否認」ツールには触れない**（2026-07-07 オーナー決定: アップロードはスキップ方針）。
- 同じURLを複数回リクエストしない（速くならない。1回で十分）。

## 注意事項

- 「インデックス登録をリクエスト」は**再クロールの保証ではなく優先度ヒント**です。
- 検査結果画面に「URLはGoogleに登録されています」と出ても、それは**旧バージョン**の話。そのまま「インデックス登録をリクエスト」を押してください。
- ライブテスト（「公開URLをテスト」）は不要です。リクエストだけでOK。
- Bing系は本日IndexNowで17 URL送信済み（全件HTTP 200/202受理）のため、この依頼書は**Google専用**です。
