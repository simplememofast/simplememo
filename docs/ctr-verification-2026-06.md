## 3. 検証手順（6/16以降）

### A. GSCから取得するデータ
1. **検索パフォーマンス → ページ** を「**過去28日**」で開き、上表の①〜⑧の各ページの **表示・クリック・CTR・平均掲載順位** を控える。
2. 期間比較を使い、**「変更前14日（〜6/1）」vs「変更後14日（6/3〜6/16）」** で各ページのCTR・順位を比較。
3. **クエリ**タブで、1ページ目（3〜9位）×0クリックのクエリ数をカウント（19→？）。
4. 会議クラスタ: `会議メモ / 議事メモ / post meeting memo / meeting memo` の平均順位（33位→？）。

→ このMDに貼ってもらえれば、私が同順位帯CTRの増減を判定します。

### B. 判定ルール（交絡を避ける）
- **同順位帯でCTRが上がったか**を見る（順位が上がってCTRが上がったのは別要因なので分離）。
- 表示数が極端に少ない（<30/28日）ページは判定保留。
- ①が **0.38%→2%** に近づけば成功＝勝ちパターン確定。

### C. 勝ったら横展開
- 効いた語順/便益の出し方を、A資料⑤（meeting-memo-template のタイトル）や他の比較・use-cases ページへ展開。
- 会議クラスタが1ページ目に入ったら、内部リンクをさらに集約。

### D. 上がらなかった場合
- タイトルを別パターン（数字/疑問形/ベネフィット先頭）でA/B。
- ①は表示2,102と大きいので、順位6.6位の底上げ（被リンク・内部リンク）も並行。
