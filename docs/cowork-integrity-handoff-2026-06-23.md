Captio後継としての被リンク獲得（作り手・メディアへの打診）にあたり、サイト上の「事実と異なる／裏付けのない記述」を是正中。**サイト側の修正（W01–W04）はコードで完了・本番反映済み**（PR #354〜#359）。本書は**リポジトリから実行できない残作業**（App Store・画像・実機計測・GSC）を担当者に引き継ぐもの。

## T1【最優先】App Store Connect 掲載文の更新
**理由:** サイトは「約1秒・ワンタップ」に修正済みだが、App Store掲載文は旧表記（0.3秒・150ms・Captio 2025年 等）の可能性。ストアとサイトの不一致は打診先（特に元Captio開発者）に即バレる。

**手順:**
1. App Store Connect → 当アプリ → 各言語ローカリゼーション（ja, en, ko, zh-Hans, zh-Hant, es, pt-BR, id, tr, ar）。
2. 「名前／サブタイトル／キーワード／説明文／プロモーションテキスト」を、リポジトリの修正済みドラフトに差し替える：
   - `docs/aso_metadata_<lang>.md`（**2026-06-23時点で下記の整合性是正済み**）
3. 反映すべき修正（旧→新）:
   - 起動 `0.3秒` → `約1秒`
   - 送信 `150ms / 150ミリ秒` → `ワンタップ`（数値は載せない）
   - `超高速` → `高速`
   - Captio終了年 `2025` → `2024年10月`
   - `最速 / 全アプリ中最速 / 10倍速い / Captioより速い` 等の優位断定 → 質的表現（スプラッシュ無し・起動直後に入力・ワンタップ送信）

**⚠️ 価格・プランは要確認（こちらで勝手に変更していない＝オーナー判断待ち）:**
- **Drafts「月額$4.99」**（ASO全言語）: サイト側のDrafts価格表記が不統一（$1.99/$3.99月・$19.99/$49.99年が混在）。Draftsの**実際の現行価格**を確認の上、ASOとサイトを1つの値に統一すること。
- **ja「7日間無料トライアル（送信無制限）」**: サイトの料金表記は「**1日3通ずっと無料**」。実際のApp Store課金体系（トライアルの有無）を確認し、ASOとサイトを一致させること。

**受け入れ基準:** ストア掲載文に `0.3秒 / 150ms / 超高速 / 最速 / 10倍 / Captio 2025` が0件。価格・無料プランの記述がサイトと一致。

## T3 OGP画像の数値焼き込み確認・作り直し
**根拠:** `docs/ogp-design-spec.md` L111 — OGテンプレのリードに **「起動 0.3 秒・送信 ワンタップ」がベタ書き**。このテンプレで生成したOG画像には `0.3秒` が焼き込まれている。

**手順:**
1. デザインソース（Figma等）のテンプレ・リードを **「起動 約1秒・送信 ワンタップ」** に修正。
2. `assets/img/og/`（全197枚）のうち、この速度リードを含むテンプレで出力された画像を再書き出し。**最低限**、主要ページのOG: `/`（home）, `/captio-alternative/`, `/blog/captio-discontinued`, `/obsidian/`, 速度系ブログ（`blog-fastest-memo-app-benchmark.png`, `blog-iphone-memo-app-fast.png` 等）。
3. 確認: 画像を開き「0.3」「150」の有無を目視、またはテキストレイヤーを検索。

**受け入れ基準:** 配信中のOG画像に `0.3秒 / 150ms` 表記が無い。

## T4【要・実機計測】最終起動値の確定
現在サイトは「**約1秒（実測中央値）**」のプレースホルダ。正確な実測値に差し替えたい。

**計測手順:**
1. 対象iPhone実機で「アプリ完全終了 → アイコンtap → 入力カーソル表示」までを**コールドスタートで5〜10回**計測。
2. **中央値**を採用（外れ値に強い）。機種・iOSバージョン・計測方法を記録。

**フィードバック:** 中央値（例「0.9秒」）＋機種・iOSを共有 → サイト全言語・`llms.txt`・ASOの「約1秒」を正確値へ一括反映する。

**補足（送信値）:** homepageのstat「150ms（送信リクエスト時間・API往復）」も、サーバーログ等で**API往復時間を実測確認**できれば数値を確定。確認できない場合はこのstat自体を削除可（未検証の数値は載せない方針）。
