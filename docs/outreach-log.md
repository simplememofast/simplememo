# Outreach Log

**Purpose**: source-of-truth log for every Reddit / HN / blog comment / email send related to Captio engagement.
**Updated by**: human, after each post via `admin/reddit/` UI (which generates the entry to paste here).

---

## Format

Each entry is a level-2 section (`## YYYY-MM-DD — DRAFT-ID (channel)`) containing the structured fields below. The admin's "Copy log entry" button generates this format automatically.

---

## Cadence summary (auto-trackable from this file)

- 1 post per day
- 3 posts per week (rolling 7d)
- 14 days minimum between same-subreddit posts

The admin UI uses browser localStorage as the live counter; this file is the durable record committed to git. If the two diverge, this file wins.
