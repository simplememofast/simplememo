### その市場固有の注意点
- インドネシア市場はApp Store課金が低い傾向があるため、無料枠（3回/日）を強調することが重要
- 「nggak」「banget」などインドネシアの口語表現を使い、親しみやすいトーンにする
- Captioの認知度はインドネシアでは低いため、「email ke diri sendiri」というコンセプト自体を説明する必要がある
- 競合はApple Notes、Google Keep、Notepadなどの無料アプリが中心。有料アプリへの抵抗感が強い市場
- インドネシアはモバイルファースト市場で、通信が不安定な地域も多いため、オフライン機能を強くアピールすべき
- ローカルアプリ（Catatan、Memo Pintar等）との差別化として「速度」と「email連携」を前面に出す
