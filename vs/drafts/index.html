シンプルメモ vs Drafts
どちらを選ぶべきかSimple Memo vs Drafts
Which Should You Choose?

SimpleMemoとDraftsの最大の違いは、送信先の柔軟性です。Draftsは「テキストが始まる場所」をコンセプトに、書いたテキストをアクション機能で様々なアプリに送信できる汎用テキストハブ。シンプルメモはメールへの送信に完全特化し、送信先を選ぶ必要のない超軽量アプリです。 The key difference is destination flexibility. Drafts is a universal text hub that sends text to any app via Actions. Simple Memo is laser-focused on email delivery — no destination to choose, just type and send in 0.3 seconds.

著者: AI Ataka · 公開日: 2026年03月06日 By AI Ataka · Published: March 6, 2026

実際にDraftsのアクション機能とシンプルメモのワンタップ送信を1週間ずつ使い比べた結果、メール送信だけが目的ならシンプルメモの方がアクション選択のステップがない分、体感で2秒以上速いことが分かりました。 After comparing Drafts' Actions and Simple Memo's one-tap send for a week each, I confirmed that if your goal is only email delivery, Simple Memo feels over 2 seconds faster by eliminating the Action selection step.

DraftsとはWhat is Drafts?

Draftsは、Agile Tortoise社が開発するApple専用のテキスト入力アプリです。「Where text starts（テキストが始まる場所）」をコンセプトに、起動と同時にカーソルが入力欄に合い、書いたテキストをアクション機能でメール、メッセージ、Twitter、Notion、Obsidianなど様々なアプリに送信できます。基本機能は無料、Pro版は月額250円（年額2,200円）です。 Drafts by Agile Tortoise is an Apple-exclusive text input app. Its concept "Where text starts" means it opens with the cursor ready. Written text can be sent to email, Messages, Twitter, Notion, Obsidian, and more via its powerful Actions system. Basic features are free; Pro costs $2.99/month ($19.99/year).

5つの決定的な違い5 Key Differences

  • 1送信先：メール専用 vs 多様な送信先
    シンプルメモはメール送信に完全特化。ワンタップで送信完了。Draftsはアクション機能で多数のアプリに送信できますが、送信先の選択が必要です。
  • 2設定：ゼロ設定 vs アクション設定
    シンプルメモはメールアドレスを1回設定するだけ。Draftsはアクションのカスタマイズが必要で、初期設定に時間がかかります。
  • 3暗号化：AES-GCM（無料） vs iCloud暗号化
    シンプルメモは端末内でAES-GCM暗号化。Draftsのテキストはアプリ内のローカル保存＋iCloud同期に依存します。
  • 4テキスト管理：送信して忘れる vs インボックス管理
    シンプルメモは送信したら完了。Draftsはテキストがインボックスに残り、処理が必要です。
  • 5Captio代替として：最適 vs 汎用
    Captioの代替を探しているなら、シンプルメモが最も近い体験を提供します。Draftsはより汎用的なテキストツールです。
  • 1Destination: Email Only vs Multiple Destinations
    Simple Memo sends to email only — one tap. Drafts can send to many apps via Actions but requires choosing a destination.
  • 2Setup: Zero vs Action Configuration
    Simple Memo needs one email address setup. Drafts requires customizing Actions, which takes time initially.
  • 3Encryption: AES-GCM (Free) vs iCloud
    Simple Memo encrypts on-device with AES-GCM. Drafts relies on local storage + iCloud sync.
  • 4Text Management: Send and Forget vs Inbox
    Simple Memo sends and you're done. Drafts keeps text in an inbox requiring processing.
  • 5Captio Alternative: Best vs General Purpose
    For Captio replacement, Simple Memo is the closest experience. Drafts is a more general text tool.

Draftsを使うべきシーンWhen to Use Drafts

シンプルメモはメール送信に特化していますが、以下のシーンではDraftsが明確に優れています。テキストを複数のアプリに振り分けたい場合、Draftsのアクション機能は非常に強力です。 Simple Memo specializes in email delivery, but Drafts is genuinely better in these scenarios. When you need to route text to multiple apps, Drafts' Action system is incredibly powerful.

  • 1複数アプリへのテキスト振り分け
    Draftsのアクション機能は、書いたテキストをNotionのデータベース、Obsidianのデイリーノート、Trelloのカード、TwitterのDMなど、あらゆるアプリに自動送信できます。テキストの送信先がメールだけでは足りない人にとって、Draftsのアクション機能は唯一無二です。
  • 2Apple Watchからの音声入力
    DraftsはApple Watchに対応しており、手首から直接音声入力が可能です。両手がふさがっている場面（料理中、運転中、ワークアウト中）でのキャプチャにはDraftsが圧倒的に便利です。シンプルメモはApple Watchには対応していません。
  • 3マークダウン編集と複雑なテキスト処理
    Draftsはマークダウンのプレビュー、構文ハイライト、カスタムキーボードバーを備えており、テキストの加工・編集にも使えます。正規表現によるテキスト変換やJavaScriptアクションなど、プログラマブルなテキスト処理が可能です。
  • 1Multi-App Text Routing
    Drafts' Action system can automatically send written text to Notion databases, Obsidian daily notes, Trello cards, Twitter DMs, and virtually any app. For those whose text destinations extend beyond email, Drafts' Actions are unmatched.
  • 2Voice Input from Apple Watch
    Drafts supports Apple Watch, enabling voice input directly from your wrist. For hands-busy situations (cooking, driving, working out), Drafts is overwhelmingly convenient. Simple Memo does not support Apple Watch.
  • 3Markdown Editing and Complex Text Processing
    Drafts offers Markdown preview, syntax highlighting, and custom keyboard bars for text editing and manipulation. Regex text transformations and JavaScript Actions enable programmable text processing that goes far beyond simple capture.

Draftsユーザーがシンプルメモへ移行するガイドMigration Guide: Drafts to Simple Memo

Draftsのアクション設定の複雑さに疲れた方、またはCaptioの代替として「メールに送るだけ」のシンプルさを求めている方向けの移行ガイドです。特にメール送信がメインユースケースの方には最適な移行先です。 This guide is for those tired of Drafts' Action configuration complexity, or seeking the simplicity of "just send to email" as a Captio replacement. If email delivery is your primary use case, Simple Memo is the ideal destination.

  • 1App Storeからシンプルメモをダウンロード
    無料版で1日3通まで送信可能。Draftsからの移行に特別な手順は不要です。インストールしてメールアドレスを設定するだけで完了します。
  • 2Draftsでの主な利用パターンを確認する
    Draftsのインボックスを見て、過去のテキストの送信先を分析します。もし80%以上がメール送信なら、シンプルメモへの移行は非常にスムーズです。他のアクション（Notion送信、Trello送信など）が多い場合はDraftsを残すことを検討してください。
  • 3シンプルメモのメール送信先を設定する
    Draftsのメールアクションで使っていたのと同じメールアドレスを設定。既存のメールフォルダ構造やフィルターはそのまま活用できます。
  • 41週間並行利用して比較する
    最初の1週間はDraftsとシンプルメモを両方インストールしたままにします。メール送信したいときはシンプルメモ、他のアクションが必要なときはDraftsと使い分けてみてください。
  • 5メール送信が主なら完全移行する
    1週間の並行利用後、Draftsのインボックスがほぼ空なら、シンプルメモへの完全移行のサインです。Draftsのサブスクリプションを解約し、シンプルメモ1本に絞ることでコストも削減できます。
  • 1Download Simple Memo from the App Store
    The free version allows up to 3 sends per day. No special migration steps from Drafts are needed. Just install and set your email address.
  • 2Review Your Usage Patterns in Drafts
    Check your Drafts inbox and analyze where you have been sending text. If over 80% goes to email, the migration to Simple Memo will be very smooth. If you frequently use other Actions (Notion, Trello, etc.), consider keeping Drafts alongside.
  • 3Configure Simple Memo's Email Destination
    Set the same email address you used in Drafts' email Action. Your existing email folder structure and filters remain fully compatible.
  • 4Run Both Apps in Parallel for One Week
    Keep both Drafts and Simple Memo installed for the first week. Use Simple Memo when you want to send email, Drafts when you need other Actions. Compare the experience directly.
  • 5Complete the Migration if Email is Primary
    After one week, if your Drafts inbox is nearly empty, that is your signal for full migration. Cancel your Drafts subscription and consolidate to Simple Memo alone, saving costs in the process.

よくある質問FAQ

Draftsとシンプルメモの最大の違いは？

Draftsは「テキストが始まる場所」をコンセプトにした汎用テキストハブで、アクション機能により多数のアプリにテキストを送信できます。シンプルメモはメール送信に完全特化した超軽量アプリです。Draftsは多機能で柔軟、シンプルメモは超特化でシンプルです。

Draftsの料金はいくらですか？

Draftsは基本機能が無料で、Pro版は月額250円（年額2,200円）です。Pro版ではアクション編集、テーマカスタマイズ、ワークスペースなどが解放されます。シンプルメモは無料で1日3通の送信が可能です。

DraftsはCaptioの代替になりますか？

Draftsでアクションを設定すればメール送信は可能ですが、Captioの「入力→自動送信」体験に最も近いのはシンプルメモです。Draftsではアクションの選択と実行が必要で、一手間多くなります。

Draftsのアクション機能とは何ですか？

アクションはDraftsの核心機能で、書いたテキストをメール、メッセージ、Notion、Obsidian、Trello、Twitter、Slackなど様々なアプリに自動送信するワークフロー自動化です。JavaScriptで独自アクションも作成可能です。非常に強力ですが、初期設定に時間がかかります。

Draftsとシンプルメモを併用する価値はありますか？

用途が重なるため、基本的には目的に合った方を1つ選ぶのがベストです。ただし、メール送信はシンプルメモ、他アプリ連携はDraftsと使い分ける上級ワークフローも可能です。

Draftsの起動速度はシンプルメモと比べてどうですか？

DraftsもAppleプラットフォーム上で非常に高速に起動し、約0.5秒で入力可能です。シンプルメモは0.3秒です。起動速度の差は小さいですが、シンプルメモはメール送信もワンタップで完了するため、キャプチャからメール送信までの総合的な所要時間ではシンプルメモが優ります。

What is the main difference between Drafts and Simple Memo?

Drafts is a universal text hub with the concept "Where text starts," capable of sending text to numerous apps via its Action system. Simple Memo is an ultra-lightweight app fully specialized for email delivery. Drafts is versatile and flexible; Simple Memo is hyper-focused and simple.

How much does Drafts cost?

Drafts' basic features are free. Pro costs $2.99/month ($19.99/year) and unlocks Action editing, theme customization, and workspaces. Simple Memo is free with 3 sends per day.

Is Drafts a good Captio alternative?

Drafts can send email via Actions, but Simple Memo provides the closest experience to Captio's "type and auto-send" flow. With Drafts, you must select and execute an Action, adding an extra step.

What exactly are Drafts Actions?

Actions are Drafts' core feature — workflow automation that sends written text to email, Messages, Notion, Obsidian, Trello, Twitter, Slack, and more. Custom JavaScript Actions are also possible. Extremely powerful, but initial setup takes time.

Is it worth using both Drafts and Simple Memo?

Since they overlap in purpose, choosing one based on your needs is generally best. However, an advanced workflow using Simple Memo for email and Drafts for other app integrations is possible.

How does Drafts' launch speed compare to Simple Memo?

Drafts also launches very quickly on Apple platforms, ready for input in about 0.5 seconds. Simple Memo launches in 0.3 seconds. While the launch speed difference is small, Simple Memo also completes email sending with one tap, making the total time from capture to email delivery faster overall.
