シンプルメモ vs Capacities
どちらを選ぶべきかSimple Memo vs Capacities
Which Should You Choose?

Capacitiesは「オブジェクト指向」のユニークなナレッジ管理ツールですが、瞬間キャプチャには構造化が重すぎます。オブジェクトタイプの選択やリレーション設定で時間がかかります。シンプルメモは「思いついた → 0.3秒で入力 → メール送信」に完全特化した、Capacitiesの高速インプットツールです。 Capacities is a unique "object-oriented" knowledge management tool — but for instant capture, its structuring requirements add friction. Choosing object types and setting relations takes time. Simple Memo is the fast "quick inbox" that captures thoughts in 0.3 seconds and emails them instantly.

CapacitiesとはWhat is Capacities?

Capacitiesは、「オブジェクト指向」のアプローチでナレッジを管理する新世代のツールです。ノートを「人」「本」「プロジェクト」などのオブジェクトタイプとして分類し、オブジェクト間のリレーションを視覚化します。デイリーノート、タグ、バックリンク、グラフビューも搭載。無料プランあり、Pro（$9.99/月）でAI機能や高度なカスタマイズが利用可能。Web・デスクトップ・モバイルに対応しています。 Capacities is a next-generation knowledge management tool using an "object-oriented" approach. Notes are classified as object types like "Person," "Book," or "Project," with visualized relations between objects. Features include daily notes, tags, backlinks, and graph view. Free plan available; Pro ($9.99/mo) adds AI features and advanced customization. Available on web, desktop, and mobile.

5つの決定的な違い5 Key Differences

  • 1起動0.3秒 vs オブジェクトタイプ選択
    Capacitiesで新規ノートを作成するには、まずオブジェクトタイプ（人・本・プロジェクトなど）を選択する必要があります。この構造化ステップが瞬間キャプチャの障壁になります。シンプルメモは起動即入力で、0.3秒でタイピングを始められます。
  • 2プレーンテキスト vs 構造化データ
    シンプルメモはプレーンテキストを入力してメールで送信するだけ。Capacitiesはオブジェクトとリレーションで知識を構造化します。素早いキャプチャにはシンプルメモ、知識の体系化にはCapacitiesが最適です。
  • 3メール配信 vs アプリ内保存
    シンプルメモで送ったメモはメール受信トレイに届き、どんなデバイスでも読めます。Capacitiesのデータはアプリ内に保存され、Capacitiesからアクセスする必要があります。
  • 4学習コストゼロ vs オブジェクト指向の理解
    シンプルメモは入力→送信のみで学習コストがゼロです。Capacitiesの「オブジェクト指向」アプローチは強力ですが、概念の理解とタイプ設計に時間がかかります。
  • 5完全オフライン vs 限定的オフライン
    シンプルメモはOutboxアーキテクチャでオフラインでも確実にメモを保存。AES-GCM暗号化で安全です。Capacitiesのオフライン機能は限定的です。
  • 10.3s Launch vs Object Type Selection
    Creating a new note in Capacities requires choosing an object type (Person, Book, Project, etc.). This structuring step creates friction for instant capture. Simple Memo opens with the cursor ready — 0.3 seconds to typing.
  • 2Plain Text vs Structured Data
    Simple Memo captures plain text and sends it via email. Capacities structures knowledge with objects and relations. For quick capture, use Simple Memo; for knowledge systematization, use Capacities.
  • 3Email Delivery vs App-Locked Content
    Memos from Simple Memo arrive in your email inbox, readable on any device. Capacities data stays within the app, requiring Capacities to access it.
  • 4Zero Learning Curve vs Object-Oriented Understanding
    Simple Memo is type-and-send with zero learning curve. Capacities' object-oriented approach is powerful but requires understanding the concepts and designing types.
  • 5Full Offline vs Limited Offline
    Simple Memo's Outbox architecture reliably saves memos offline with AES-GCM encryption. Capacities' offline capabilities are limited.

よくある質問FAQ

Capacitiesとは何ですか？

Capacitiesは「オブジェクト指向」のナレッジ管理ツールです。ノートを「人」「本」「プロジェクト」などのタイプに分類し、オブジェクト間の関係を視覚化します。PKMに特化した新しいアプローチのツールです。

Capacitiesの無料プランはどこまで使えますか？

Capacitiesの無料プランでは基本的なオブジェクト管理、デイリーノート、バックリンクが利用可能です。Pro（$9.99/月）でAI機能、高度なカスタマイズ、優先サポートが追加されます。

シンプルメモとCapacitiesは併用できますか？

最適な組み合わせです。シンプルメモで瞬間的にアイデアをキャプチャしてメールで受信し、時間がある時にCapacitiesの適切なオブジェクトタイプとして整理するワークフローが効果的です。

Capacitiesの学習コストは高いですか？

「オブジェクト指向」の概念理解とオブジェクトタイプの設計に時間がかかります。PKMに真剣に取り組む人には価値がありますが、素早いメモだけなら学習コストゼロのシンプルメモがおすすめです。

Capacitiesはオフラインで使えますか？

Capacitiesのオフライン機能は限定的です。シンプルメモはOutboxアーキテクチャにより、オフラインでもAES-GCM暗号化でメモを保存し、接続回復時に自動送信します。

CapacitiesとNotionの違いは何ですか？

Notionはページベースの汎用ワークスペースで、データベース・ドキュメント・タスク管理を統合します。Capacitiesはオブジェクト指向のPKM専用ツールで、ノートを型付きオブジェクトとして管理します。PKMに集中するならCapacities、多目的ツールが必要ならNotionが適しています。SimpleMemoはどちらとも併用できるキャプチャ専用ツールです。

Capacitiesのデータはエクスポートできますか？

はい、CapacitiesからMarkdown形式でデータをエクスポート可能です。オブジェクトタイプの構造も保持されます。SimpleMemoのメモはメールとして残るため、エクスポートの必要自体がありません。

Capacitiesのモバイルアプリは使いやすいですか？

Capacitiesにはモバイルアプリがあり、デスクトップ版と同期します。ただし、オブジェクトタイプの選択やリレーション設定はスマートフォンではやや操作しにくい部分があります。モバイルでの素早いキャプチャにはSimpleMemoの方が快適です。

Capacitiesの日本語対応はどうですか？

Capacitiesのインターフェースは主に英語ですが、日本語コンテンツの入力・表示は問題ありません。ただし、ヘルプドキュメントやコミュニティは英語が中心です。SimpleMemoは日本語・英語の両方に完全対応しており、日本語ユーザーにも安心です。

Capacitiesの無料プランからProにアップグレードする価値はありますか？

AI機能、高度なカスタマイズ、優先サポートが必要ならProは価値があります（$9.99/月）。ただし、基本的なオブジェクト管理とデイリーノートは無料プランで十分です。素早いキャプチャが主な目的なら、SimpleMemoの無料プラン（1日3通）の方がコスト効率が高いです。

What is Capacities?

Capacities is an "object-oriented" knowledge management tool. Notes are classified as types like Person, Book, or Project, with visualized relations between objects. It's a novel approach to PKM.

What can I do with Capacities' free plan?

The free plan includes basic object management, daily notes, and backlinks. Pro ($9.99/mo) adds AI features, advanced customization, and priority support.

Can I use Simple Memo alongside Capacities?

This is the optimal combination. Capture ideas instantly with Simple Memo, receive them via email, then organize into the appropriate Capacities object types when you have time.

Is Capacities hard to learn?

Understanding the object-oriented concept and designing object types takes time. It's valuable for serious PKM practitioners, but for quick notes, Simple Memo's zero learning curve is better.

Does Capacities work offline?

Capacities' offline capabilities are limited. Simple Memo's Outbox architecture saves memos offline with AES-GCM encryption and auto-sends when connectivity returns.

How does Capacities compare to Notion?

Notion is a page-based general-purpose workspace integrating databases, documents, and task management. Capacities is an object-oriented PKM-specific tool managing notes as typed objects. For focused PKM, choose Capacities; for multi-purpose needs, Notion. SimpleMemo works alongside either as a dedicated capture tool.

Can I export data from Capacities?

Yes, Capacities supports Markdown export with object type structures preserved. SimpleMemo notes remain as emails, so export isn't needed at all.

Is Capacities' mobile app user-friendly?

Capacities has a mobile app that syncs with desktop. However, object type selection and relation setup can feel cumbersome on phones. For quick mobile capture, SimpleMemo is more comfortable.

Does Capacities support Japanese?

Capacities' interface is primarily English, but Japanese content input and display work fine. Help documentation and community are English-focused. SimpleMemo fully supports both Japanese and English.

Is upgrading from Capacities free to Pro worth it?

Pro is valuable if you need AI features, advanced customization, and priority support ($9.99/mo). Basic object management and daily notes work well on free. If quick capture is your main goal, SimpleMemo's free plan (3 sends/day) is more cost-effective.

Capacitiesを使うべきケースWhen to Use Capacities

公平を期すために、Capacitiesの方が明らかに優れている場面を紹介します。以下のような用途がメインなら、Capacitiesを選ぶべきです。 In fairness, here are scenarios where Capacities is genuinely the better choice. If these match your primary use cases, Capacities is the right tool.

  • 1ノート間の関係性を可視化したい
    Capacitiesのオブジェクト指向アプローチでは、「人」「プロジェクト」「本」などのオブジェクトタイプを定義し、それらの関係（リレーション）を視覚的に把握できます。グラフビューで全体像を俯瞰し、知識の構造を理解するのに最適です。SimpleMemoにはノート間のリンクや関係性の機能はありません。
  • 2デイリーノートを習慣にしたい
    Capacitiesのデイリーノート機能は、毎日の記録を自動的に作成し、バックリンクでオブジェクトに関連付けます。日記、振り返り、ジャーナリングを習慣化したい方に最適です。SimpleMemoは個別メモの即時送信に特化しており、継続的な日次記録には向いていません。
  • 3Notion風のUIでPKMを始めたい
    Capacitiesの直感的なインターフェースはNotionに近い使い心地で、PKM初心者にもとっつきやすいです。TanaやObsidianと比べて視覚的に洗練されており、学習コストも中程度に抑えられています。SimpleMemoはPKMツールではなくキャプチャ専用ツールです。
  • 4AI機能でノートを拡張したい
    Capacities Proの AI機能では、ノートの要約、関連コンテンツの提案、文章の拡張などが可能です。ナレッジ管理にAIの力を借りたい場合、Capacitiesは優れた選択肢です。SimpleMemoにはAI機能は搭載されていません。
  • 1Visualizing relationships between notes
    Capacities' object-oriented approach lets you define types like "Person," "Project," "Book" and visualize their relations. Graph view provides a bird's-eye view of your knowledge structure. SimpleMemo has no linking or relationship features.
  • 2Building a daily notes habit
    Capacities' daily notes automatically create daily entries with backlinks to objects. Perfect for journaling, reflection, and daily logging. SimpleMemo is built for instant individual memo delivery, not ongoing daily records.
  • 3Starting PKM with a Notion-like UI
    Capacities' intuitive interface feels similar to Notion, making it approachable for PKM beginners. Visually polished compared to Tana or Obsidian, with moderate learning curve. SimpleMemo is a capture tool, not a PKM platform.
  • 4Enhancing notes with AI
    Capacities Pro's AI features offer note summarization, related content suggestions, and text expansion. For AI-assisted knowledge management, Capacities is an excellent choice. SimpleMemo has no AI features.

CapacitiesからSimpleMemoへの移行ガイドMigration Guide: Capacities to SimpleMemo

Capacitiesのオブジェクト選択の手間に疲れた方や、もっとシンプルにメモを取りたい方向けのガイドです。CapacitiesとSimpleMemoの併用が最も効果的ですが、シンプル化したい場合の手順も紹介します。 This guide is for those tired of Capacities' object selection overhead or wanting a simpler note-taking approach. Using both together is most effective, but here are steps for simplification.

  • 1SimpleMemoをインストールしてメールアドレスを設定
    App StoreからSimpleMemoをダウンロード。メールアドレスを設定するだけで完了。アカウント作成は不要。Capacitiesのオブジェクトタイプ設計やワークスペース構築と比べて、セットアップは30秒で終わります。
  • 2Capacitiesのデータをエクスポート
    Capacitiesの設定画面からデータをエクスポートします。Markdown形式で出力可能です。オブジェクトタイプの構造も保持されるため、将来Capacitiesに戻りたくなった場合も安心です。
  • 3素早いメモはSimpleMemoで、構造化はCapacitiesで
    オブジェクトタイプの選択が不要な場面（アイデアの断片、買い物メモ、ちょっとした思いつき）はSimpleMemoで0.3秒入力→メール送信。構造化が必要なもの（プロジェクト管理、読書ノート、人物データベース）はCapacitiesで管理する、という使い分けが最適です。
  • 4週次でメール受信トレイを整理
    SimpleMemoで1週間送信したメモを週末にまとめて確認。重要なものはCapacitiesの適切なオブジェクトタイプに転記し、それ以外はメールのアーカイブで保管します。この方法なら、Capacitiesの構造化の恩恵を受けつつ、日常のキャプチャ速度を犠牲にしません。
  • 1Install SimpleMemo and set your email address
    Download SimpleMemo from the App Store. Just set your email — no account creation needed. Compared to designing Capacities object types and building workspaces, setup takes 30 seconds.
  • 2Export your Capacities data
    Export from Capacities settings in Markdown format. Object type structures are preserved, so you can return to Capacities later if needed.
  • 3Quick notes via SimpleMemo, structured work via Capacities
    For moments that don't need object types (idea fragments, shopping lists, quick thoughts), use SimpleMemo's 0.3-second input and email delivery. For structured work (project management, reading notes, people databases), use Capacities. This division is optimal.
  • 4Weekly inbox review and organization
    Review the week's SimpleMemo emails on weekends. Transfer important items to the right Capacities object types; archive the rest. This preserves Capacities' structuring benefits without sacrificing daily capture speed.
