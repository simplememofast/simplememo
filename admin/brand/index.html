この画面は /admin/* の Basic 認証で保護されています。外部公開されません。 すべてのテストは Cloudflare Pages Functions 経由で実行され、 X API キーはブラウザには送られません。

1. 資料ファイル疎通確認

ブランドブック・X 運用ガイド・セットアップ手順の Markdown ファイルが、認証付きでこの画面から読み取れるかを確認します。

2. Cloudflare 環境変数の状態

X API 関連の Secret が Cloudflare Pages に登録されているかを確認します。値は表示されません。設定済みか未設定かだけが返ります。

3. X API 接続テスト（検索）

X_BEARER_TOKEN を使って X API v2 の GET /2/tweets/search/recent にリクエストします。

※ このエンドポイントは X の Basic ティア以上のプランが必要です。Free ティアでは 403 が返ります。

4. 自アカウント情報取得テスト

X_USER_ACCESS_TOKEN（OAuth 2.0 User Context）が登録されている場合、GET /2/users/me を叩いて自アカウント情報を取得します。App-only Bearer Token では取得できないエンドポイントです。

5. 認証情報の再入力が必要な場合

Basic 認証の Cookie を削除してログインし直したい場合は、ブラウザのプライバシー設定からこのサイトの Cookie を削除してください。

X 検索 API（v2 search/recent または search/all）で captio のメモ帳アプリに言及している日本語ツイートを 1 コールずつ取得します。 課金を抑えるため ページングは手動（「次のページを取得」ボタンを押した時だけ API を叩きます）。 取得済みツイートはブラウザ内（localStorage）に保存され、同じ ID は重複追加されません。

収集条件

※ min_faves: などの高度オペレータは X のプラン／エンドポイントによって挙動が異なります。search/all（Pro 以上）推奨。

※ 現プランでは X API の min_faves: 演算子が使えないため、受信後にクライアント側で間引きます。API 課金は取得件数分かかります。
